# CacheSpec
Model level caching framework for Ruby on Rails 3 or up.

## Installation

Add this line to your application's Gemfile:

    gem 'cache_spec'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install cache_spec

## Development Tasks
